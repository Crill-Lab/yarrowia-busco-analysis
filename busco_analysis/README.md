# Y. lipolytica BUSCO Analysis Pipeline

BUSCO quality assessment pipeline for Y. lipolytica genome assemblies developed for the DARPA TOGLE project.

## Overview
- 67 Y. lipolytica strains analyzed
- Saccharomycetes_odb10 database (2,137 BUSCOs)
- Average completion: 91.0%
- Interactive visualizations included
